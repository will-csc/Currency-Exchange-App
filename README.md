# Currency Converter App

A modern and intuitive currency converter application built with React Native and Expo.

## Features

- **Real-time Conversion**: Instantly convert values between different currencies.
- **Modern UI**: Clean, dark-themed interface with intuitive color coding (Blue for "From", Green for "To").
- **Currency Swap**: Easily swap source and target currencies with a single tap.
- **Multiple Currencies**: Supports major currencies including USD, BRL, EUR, GBP, JPY, CAD, AUD, and CHF.
- **Responsive Design**: Works smoothly on both iOS and Android devices.

- **Expo**: For easy development, build, and deployment.
- **JavaScript (ES6+)**: Core logic and state management.

## Installation and Running

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/will-csc/conversor-app.git
    cd conversor-app
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the application:**
    ```bash
    npx expo start
    ```

4.  **Open on your device:**
    - Scan the QR code with the **Expo Go** app (available on App Store and Google Play).
    - Or press `a` for Android Emulator / `i` for iOS Simulator.
